#  A Rule-Constrained Decision Framework Implemented on CARLA

+ This repository provides a partial implementation of GAI on CARLA.


+ Installing necessary packages:
   + python: 3.7.16
   + carla: 0.9.15

+ Usage:
   + Put the files in the folder ``.../CARLA_0.9.15/PythonAPI/examples``.
   + Run ``automatic_control_RuleEngine.py``.
